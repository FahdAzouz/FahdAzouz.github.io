BustTheGhost

AI-driven game using Bayesian inference for ghost location estimation, improving prediction accuracy by 35%.

Key Features:

  • BustTheGhost is an innovative AI-driven game that leverages Bayesian inference to create an engaging ghost-hunting experience. The project demonstrates the practical application of probabilistic reasoning in a fun, interactive setting. Key features and accomplishments: 1. Bayesian Inference: Implemented a Bayesian inference system to update probabilities of the ghost's location based on user actions and sensor readings. 2. Dual Sensor System: Developed two independent sensors - a distance sensor and a direction sensor - each providing different types of evidence about the ghost's location. 3. Dynamic Probability Updates: Created a system that dynamically updates and displays probabilities on the game grid, giving players real-time feedback on their ghost-hunting progress. 4. User Interface: Designed an intuitive web-based interface using HTML, CSS, and JavaScript, allowing players to interact with the game board and view updated probabilities. 5. Algorithm Optimization: Through iterative refinement, improved the ghost location prediction accuracy by 35%, enhancing the game's challenge and realism. 6. Educational Value: The project serves as an interactive demonstration of Bayesian statistics, probability theory, and decision-making under uncertainty. This project showcases my skills in AI development, probabilistic modeling, web development, and game design. It demonstrates my ability to apply complex mathematical concepts to create engaging, user-friendly applications.
